## Suite Intent

This suite turns the UI/UX roadmap into executable Codex skills. Each skill is deliberately scoped so the transformation can proceed in controlled phases without broad unrelated rewrites.

The north star is:

> Here is the state. Here is the risk. Here is the action. Here is the proof.

## Shared Non-Goals

- Do not rebuild unrelated backend services.
- Do not create a parallel theme system.
- Do not add decorative UI effects to simulate quality.
- Do not broaden a phase into unrelated payroll, seed, or module work.
- Do not make dashboard pages client-compute business truth.
- Do not skip RBAC, tenant isolation, module entitlement, redaction, audit, or safe error handling.
