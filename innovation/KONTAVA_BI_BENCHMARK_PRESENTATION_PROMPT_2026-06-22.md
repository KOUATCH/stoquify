# KONTAVA BI Benchmark Presentation Prompt

Date: 2026-06-22

## Full Professional Prompt

Act as a senior business intelligence strategist, product architect, and enterprise platform consultant.

Your task is to research and analyze 5 of the best modern business platforms that use Business Intelligence deeply to improve user outcomes, operational efficiency, revenue growth, decision-making, and platform trust.

Focus on platforms such as enterprise SaaS, commerce, CRM, accounting, logistics, fintech, ERP, or marketplace platforms. Select platforms that clearly use BI in practical, user-facing and internal decision-making ways.

For each platform, investigate:

1. What business intelligence capabilities they use
2. How BI improves the user experience
3. How BI supports management, operations, revenue, risk, and growth
4. What dashboards, analytics, automation, forecasting, alerts, scoring, or recommendations they provide
5. What lessons can be adapted for KONTAVA

Then produce a professional presentation for KONTAVA with the following structure:

## Presentation Title

10 High-Value Business Intelligence Capabilities KONTAVA Should Adopt from Leading Business Platforms

## Required Sections

### 1. Executive Summary

Explain why BI is critical for KONTAVA and how it can become a strategic advantage for users, managers, partners, and decision-makers.

### 2. Benchmark Platforms Studied

Present the 5 selected platforms and briefly explain why each was chosen.

For each platform include:

- Platform name
- Industry/category
- Key BI strengths
- Type of users served
- Most relevant lessons for KONTAVA

### 3. The 10 Most Valuable BI Aspects

Identify the 10 most valuable BI practices found across the benchmark platforms.

For each BI aspect, explain:

- What it is
- How leading platforms use it
- Why it creates business value
- Where KONTAVA can use it
- Which KONTAVA users benefit from it
- Required data sources
- Suggested dashboards, alerts, or workflows
- Implementation priority: High, Medium, or Low

Cover areas such as:

- Executive dashboards
- Sales and revenue analytics
- Customer behavior intelligence
- Inventory or service performance analytics
- Financial health monitoring
- Forecasting and predictive analytics
- Risk and anomaly detection
- User performance scorecards
- Operational bottleneck tracking
- Personalized recommendations and decision support

### 4. KONTAVA Application Blueprint

Map the BI capabilities into KONTAVA modules or product areas.

Use a table with:

- BI capability
- KONTAVA module
- User group
- Data needed
- Business value
- Implementation phase

### 5. Recommended BI Dashboard Suite for KONTAVA

Design a practical BI dashboard suite, including:

- Founder/CEO dashboard
- Finance dashboard
- Sales dashboard
- Operations dashboard
- Customer success dashboard
- Risk/compliance dashboard
- User self-service analytics dashboard

For each dashboard, list:

- Main metrics
- Visualizations
- Alerts
- Decisions it supports

### 6. Implementation Roadmap

Create a phased roadmap:

- Phase 1: Core reporting foundation
- Phase 2: Operational dashboards
- Phase 3: Predictive analytics and alerts
- Phase 4: AI-assisted recommendations
- Phase 5: Embedded BI monetization

### 7. Strategic Recommendations

Conclude with clear recommendations on how KONTAVA should use BI to become more intelligent, user-centered, scalable, and competitive.

## Output Format

Produce the answer as a professional slide-style presentation with:

- Slide titles
- Concise bullet points
- Tables where helpful
- Practical examples
- Clear KONTAVA-specific recommendations
- Source references for researched platforms

Make the presentation modern, enterprise-grade, strategic, and directly useful for product planning, investor communication, and implementation planning.

## Short Version

Research 5 leading business platforms that use Business Intelligence extensively to improve user outcomes, platform operations, revenue, risk control, and decision-making. From that research, create a professional presentation identifying the 10 most valuable BI capabilities KONTAVA should adopt.

For each BI capability, explain what it is, how top platforms use it, why it matters, where KONTAVA can apply it, which users benefit, what data is required, and how it should appear in dashboards, alerts, workflows, or recommendations.

Deliver the result as a slide-style enterprise presentation with benchmark findings, a KONTAVA BI blueprint, dashboard recommendations, and a phased implementation roadmap.
